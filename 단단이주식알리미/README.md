# 단단이 주식 알리미

SK하이닉스·삼성전자·KODEX 미국원자력SMR·한전KPS 등 관심 종목의 **실시간 시세**를 한 화면에서 보고,
등락률에 따라 **7단계 색상**으로 강조한다. 각 종목에 대해 **증설·해외진출·신규고객/수주** 키워드
뉴스가 나오면 화면 뉴스 패널에 모아 보여주고 새 기사는 **NEW** 로 강조한다.

## 기능
- 실시간 시세(15초 폴링) — 네이버 금융 데이터
- 전일 종가 대비 등락률 7단계 색상: `+10%↑ / +5~10% / 0~+5% / 보합 / 0~-5% / -5~-10% / -10%↓`
  (±10% 도달 시 테두리 + 펄스로 초강조)
- 종목 추가/삭제 (이름 검색 또는 종목코드 입력, 목록은 브라우저에 저장)
- 키워드 뉴스 패널(4분 폴링) + 키워드 추가/삭제 + 신규 기사 NEW 강조

## 로컬 실행
```powershell
npm i -g vercel      # 최초 1회
vercel dev           # http://localhost:3000
```

## 배포 (Vercel 무료)
```powershell
vercel               # 안내 따라 프로젝트 연결
vercel --prod        # 운영 배포
```
또는 이 폴더를 GitHub에 올리고 vercel.com에서 Import.

## (선택) 네이버 뉴스 API 키
키를 넣으면 한국 뉴스 품질이 더 좋아진다. **없으면 Google News RSS로 자동 대체**된다.
1. https://developers.naver.com → 애플리케이션 등록 → "검색" API 사용
2. Vercel 프로젝트 → Settings → Environment Variables 에 추가:
   - `NAVER_CLIENT_ID`
   - `NAVER_CLIENT_SECRET`
3. 재배포(`vercel --prod`)

## ⚠️ 주의
- 네이버 **시세** 엔드포인트는 비공식이다. 개인/내부용 폴링은 통상 문제되지 않으나
  약관상 회색지대이며 네이버가 차단/변경하면 `api/quotes.js`의 엔드포인트를 수정해야 한다.
  상업적 대량 호출은 금지. 안정성이 중요해지면 한국투자증권 Open API(공식)로 교체 권장.
- 표시 시세는 지연될 수 있다. 투자 판단의 유일한 근거로 삼지 말 것.
